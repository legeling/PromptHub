# 快速开始

欢迎使用 PromptHub！本指南将帮助你在 2 分钟内开始管理你的 Prompt。

## 1. 安装与启动

根据你的操作系统下载并安装，首次启动后你将看到一个清晰的工作区。

- **macOS**: 下载 DMG 并拖入 Applications。
- **Windows**: 运行安装程序即可。

## 2. 视觉化管理

![画廊模式](/imgs/2-gallery-view.png)

PromptHub 默认提供**画廊模式**，让你通过卡片一眼找到所需的提示词。你也可以在右上角切换到列表模式进行批量操作。

## 3. 创建第一个 Prompt

1. 点击右上角的 **"+ 新建"**。
2. 输入标题并编写提示词内容。
3. 尝试使用 `{{variable}}` 语法，保存后它将变成可填写的表单。
4. 按 `Cmd/Ctrl + S` 快速保存。

## 4. 变量与使用

在 Prompt 详情页，点击底部的 **"复制 Prompt"**。如果包含变量，系统会提示你填写。填写完成后，渲染后的内容将自动进入剪贴板。

## 5. 必备快捷键

我们精简了最常用的几个操作：

| 快捷键             | 功能                |
| :----------------- | :------------------ |
| `Alt + Shift + P`  | 快速显示 / 隐藏窗口 |
| `Alt + Shift + N`  | 快速新建 Prompt     |
| `Cmd/Ctrl + F`     | 全局全文搜索        |
| `Cmd/Ctrl + Enter` | 保存并关闭编辑器    |
