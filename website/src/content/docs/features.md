# 核心功能

PromptHub 是一款功能丰富的 AI Prompt 管理工具。本页详细介绍各项核心功能。

## Prompt 管理

### 创建与编辑

![主界面](/imgs/1-index.png)

- **富文本编辑器**：支持 Markdown 语法高亮
- **实时预览**：编辑时即时预览最终效果
- **自动保存**：编辑内容自动保存为草稿

### 文件夹组织

按项目或用途组织你的 Prompt：

- 支持无限层级的文件夹结构
- 拖拽排序和移动
- 批量操作（移动、删除）

### 标签系统

使用标签进行灵活分类：

- 为每个 Prompt 添加多个标签
- 按标签快速筛选
- 标签颜色自定义

### 视图模式

![画廊视图](/imgs/2-gallery-view.png)
![列表视图](/imgs/3-list-view.png)

PromptHub 提供三种视图模式（卡片、画廊、列表），适配不同的使用场景。

## 变量模板

![变量填充](/imgs/7-variable.png)

### 基础语法

使用双花括号定义变量：

```text
你是一位 {{role}} 专家。
请帮我分析以下内容：
{{content}}
```

### 变量类型

| 类型   | 语法                    | 说明         |
| ------ | ----------------------- | ------------ |
| 文本   | `{{name}}`              | 单行文本输入 |
| 长文本 | `{{content:textarea}}`  | 多行文本输入 |
| 选择   | `{{type:select:A,B,C}}` | 下拉选择框   |

## 双语对照模式

![双语对照](/imgs/6-double-language.png)

PromptHub 支持为同一个 Prompt 维护中英文两个版本：

- **一键切换**：在详情页快速切换中英文版本
- **智能适配**：复制或进行 AI 测试时，自动使用当前显示的语言版本
- **翻译辅助**：方便进行提示词的国际化调优

## 版本控制

### 自动版本

每次保存自动创建版本快照：

- 记录修改时间
- 保存完整内容
- 支持版本注释

### 版本对比

![版本对比](/imgs/8-version-compare.png)

- 并排对比两个版本
- 高亮显示修改内容
- 查看新增/删除的行

### 版本回滚

一键恢复到任意历史版本，当前版本会自动备份。

## 多模型测试

### 支持的模型

- **OpenAI**: 支持 GPT 全系列模型
- **Anthropic**: 支持 Claude 全系列模型
- **Google**: 支持 Gemini 全系列模型
- **主流国产及开源模型**: 支持 DeepSeek, 通义千问, 智谱 AI, 文心一言等及本地模型 (如 Ollama)

### 配置 API

进入 **设置 → AI 模型**：

1. 选择服务商
2. 输入 API Key
3. 可选：配置自定义 Base URL

### 并行测试

![并行测试](/imgs/6-double-language.png)

1. 打开 Prompt 详情页
2. 点击底部的 **"AI 测试"** 按钮
3. 选择要对比的模型（最多 4 个）
4. 填写变量并点击 **"运行"**

测试结果会并排显示，方便对比不同模型的回复质量。

## 搜索功能

### 全文搜索

基于 SQLite FTS5 的高性能全文搜索：

- 支持中英文分词
- 毫秒级响应
- 搜索标题、内容、标签

### 快捷搜索

按 `Cmd/Ctrl + F` 打开全局搜索：

- 实时搜索建议
- 键盘导航
- 快速跳转

## 数据导出

### 导出格式

- **JSON**: 完整数据，适合备份
- **YAML**: 可读性高，适合版本控制
- **CSV**: 表格格式，适合分析

### 导出选项

- 导出单个 Prompt
- 导出整个文件夹
- 导出全部数据

![数据备份](/imgs/4-backup.png)

## 数据同步

### WebDAV 同步

支持任何 WebDAV 服务：

- Nextcloud
- Synology NAS
- 坚果云
- 自建服务器

### 同步设置

1. 进入 **设置 → 同步**
2. 输入服务器地址
3. 填写用户名和密码
4. 选择同步频率

### 冲突处理

- 自动检测冲突
- 保留双方版本
- 手动选择保留版本

## 高级设置

PromptHub 提供了丰富的自定义选项，你可以根据自己的习惯进行配置：

### 个性化显示

![主题设置](/imgs/5-theme.png)

- **主题颜色**：内置 6 种莫兰迪色系主题，并支持自定义 HEX 颜色。
- **字体大小**：提供小、中、大三种字号，适配不同的屏幕分辨率。
- **深色模式**：支持跟随系统、始终开启或始终关闭。
- **Markdown 渲染**：可全局开启或关闭详情页的 Markdown 实时渲染。

### 编辑习惯

- **自动保存**：编辑期间自动保存草稿，防止数据意外丢失。
- **显示行号**：在编辑器中显示行号，方便定位。
- **编辑器预览**：默认在新打开编辑器时显示预览模式。

### 系统集成 (Windows)

- **关闭行为**：可设置点击关闭按钮时是最小化到托盘还是彻底退出。
- **开机启动**：支持登录系统时自动启动并最小化运行。

### AI 参数优化

为不同的任务场景配置专属参数：

- **对话模型**：调节 Temperature、Max Tokens、Frequency Penalty 等。
- **思考模式**：支持 DeepSeek-V3 等思考型模型的推理过程显示。
- **图像模型**：配置生成尺寸、质量（Standard/HD）和风格（Vivid/Natural）。

## 快捷键参考

为了提高效率，PromptHub 提供了丰富的全局和本地快捷键支持。

### 常用快捷键

| 快捷键             | 功能                |
| :----------------- | :------------------ |
| `Alt + Shift + P`  | 快速显示 / 隐藏窗口 |
| `Alt + Shift + N`  | 快速新建 Prompt     |
| `Cmd/Ctrl + F`     | 全局全文搜索        |
| `Cmd/Ctrl + S`     | 保存编辑器内容      |
| `Cmd/Ctrl + Enter` | 保存并关闭编辑器    |
| `Escape`           | 清除搜索 / 关闭弹窗 |
