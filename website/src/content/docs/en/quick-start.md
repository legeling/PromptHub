# Quick Start

Welcome to PromptHub! This guide will help you get started in under 2 minutes.

## 1. Install & Launch

Download and install the app for your OS. You'll see a clean workspace on first launch.

- **macOS**: Download DMG and drag to Applications.
- **Windows**: Run the installer.

## 2. Visual Management

![Gallery Mode](/imgs/2-gallery-view.png)

PromptHub defaults to **Gallery Mode**, allowing you to find prompts at a glance with visual cards. You can also switch to list view in the top right for bulk actions.

## 3. Create First Prompt

1. Click **"+ New"** in the top right.
2. Enter a title and write your prompt content.
3. Try using `{{variable}}` syntax; it becomes a fillable form after saving.
4. Press `Cmd/Ctrl + S` to save instantly.

## 4. Variables & Usage

In the prompt details, click **"Copy Prompt"** at the bottom. If variables are present, you'll be prompted to fill them. The rendered content is automatically copied to your clipboard.

## 5. Essential Shortcuts

We've streamlined the most useful operations:

| Shortcut           | Action                  |
| :----------------- | :---------------------- |
| `Alt + Shift + P`  | Show / Hide Window      |
| `Alt + Shift + N`  | Create New Prompt       |
| `Cmd/Ctrl + F`     | Global Full-Text Search |
| `Cmd/Ctrl + Enter` | Save & Close Editor     |
