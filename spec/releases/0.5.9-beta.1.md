# PromptHub 0.5.9-beta.1 Release Record

## Release

- Version: `0.5.9-beta.1`
- Tag: `v0.5.9-beta.1`
- Date: 2026-06-14
- Channel: Preview / beta

## Summary

`0.5.9-beta.1` is a preview release focused on MCP management, prompt relationship structure, and Skill import/preview ergonomics.

## User-Facing Highlights

- MCP management workspace preview, including local MCP library, official template store, agent-target distribution, static health checks, selective `.env` import, conflict protection, and CLI MCP commands.
- Prompt relationship tree and semantic relations in the existing prompt workflow, including drag-to-group parent/child prompts, expand/collapse controls, parent labels, child counts, and detail-page relationship navigation.
- Git Skill import improvements, including local clone-based SSH GitHub scanning, stale-result reset after URL changes, rescan entry points, larger result scrolling, and clearer HTTPS rate-limit guidance.
- Skill image resource preview improvements, including wheel zoom, grab-to-pan, fixed bottom-right controls, fullscreen preview, Escape close, and no extra shadow/frame treatment.
- Skill detail polish, including visible versions starting at `v1` and click-to-copy title behavior.

## Release Surface Decisions

- Package versions are bumped to `0.5.9-beta.1`.
- `CHANGELOG.md`, website changelog, website generated release metadata, root README, and localized README files include the `0.5.9-beta.1` preview summary.
- Stable download links in README files continue to point to `v0.5.8` stable, while the preview row points to the explicit `v0.5.9-beta.1` prerelease tag.
- Screenshots were not regenerated for this beta; README screenshot notes clarify that the current images still show the stable `v0.5.8` surfaces with beta additions described in text.
