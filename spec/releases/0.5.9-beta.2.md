# PromptHub 0.5.9-beta.2 Release Record

## Release

- Version: `0.5.9-beta.2`
- Tag: `v0.5.9-beta.2`
- Date: 2026-06-27
- Channel: Preview / beta

## Summary

`0.5.9-beta.2` is a preview release focused on Plugin management, full Agent asset sync, network configuration, CLI ergonomics, Skill source update correctness, and MCP / Plugin UI parity with Skill surfaces.

## User-Facing Highlights

- Expanded Plugin management preview, including My Plugins, Plugin Store, Agent Plugin, version snapshots, source updates, batch actions, Agent distribution, child Skill / MCP import, and static safety assessment.
- Full Agent asset sync across self-hosted sync and backup/restore for My Skills, My MCP, My Plugins, Rules, and related data.
- Network settings category for proxy and mirror-source configuration, with network requests using a shared proxy policy.
- CLI project workflow improvements, including current-directory installs, interactive My Skill selection, path installs, and AI configuration commands.
- More reliable Skill source update checks through content fingerprints and directory comparison that ignore caches, generated artifacts, hidden files, and PromptHub metadata.
- MCP / Plugin UI alignment with Skill patterns for tag filtering, detail pages, card widths, update badges, and distribution entry points.
- Windows Agent path compatibility for `%LOCALAPPDATA%`, Hermes, Trae Work, and shell-open path handling.
- Self-hosted Web captcha can be disabled by environment variable for private deployments.
- Release pipeline guards optional mirror publishing for prerelease workflows.

## Release Surface Decisions

- Package versions are bumped to `0.5.9-beta.2`.
- `CHANGELOG.md`, website changelog, website generated release metadata, root README, and localized README files include the `0.5.9-beta.2` preview summary.
- Stable download links in README files continue to point to `v0.5.8` stable, while the preview row points to the explicit `v0.5.9-beta.2` prerelease tag.
- Screenshots were not regenerated for this beta; the release includes substantial UI polish, but no screenshot refresh was requested for this release pass.
